Democratizing Disaster Surveillance

An open-source UAV framework utilizing Computational Ghost Imaging.

Version: 1.0.0-beta

About the Framework

The CGI-UAV framework is designed from the ground up for embedded systems and Edge AI efficiency. By utilizing Computational Ghost Imaging, this drone platform can reconstruct high-fidelity visuals in visually degraded environments where traditional optics fail.

Quick Start

Clone the repository and install the required edge-computing dependencies for the CGI pipeline.

$ git clone https://github.com/redfox1111111/CGI-OpenSource-Framework.git
$ cd CGI-OpenSource-Framework
$ pip install -r requirements.txt
$ python3 reconstruct.py --svd true
